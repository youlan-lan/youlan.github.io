  • git 命令

    基本

    • 创建文件夹

      $ mkdir learngit
      $ cd learngit
      $ pwd //显示当前目录路径

    • git init 管理仓库

    • git add 添加到仓库

    • git commit -m

    • git status 看状态

    • git status告诉你有文件修改过，用git diff可以查看修改内容

    • git log 查看提交历史 “commit id”

    • git reflog 查看命令历史

      上一个版本就是HEAD^，上上一个版本就是HEAD^^……

    • git reset HEAD 将暂存区的修改撤销，回到工作区

    • git checkout –file 将工作区的文件删除

    • git rm 删除文件

    远程仓库

    • id_rsa.pub公钥添加到GitHub上，

    • $ git remote add origin git@github.com:xxx/xxx.git

    • $ git push -u origin master

      $ git push origin master

    • git clone 克隆

    标签

    • git tag 新建标签，默认为HEAD，也可以指定一个commit id
    • git tag -a -m “blablabla…”指定标签信息
    • git tag 查看所有标签
    • git push origin 推送一个本地标签
    • git push origin –tags 推送全部未推送过的本地标签
    • git tag -d 删除一个本地标签
    • git push origin :refs/tags/ 删除一个远程标签

    分支

    • 查看分支：git branch
    • 创建分支：git branch
    • 切换分支：git checkout 或者 git switch
    • 创建+切换分支：git checkout -b 或者 git switch -c
    • 合并某分支到当前分支：git merge
    • 删除分支：git branch -d
